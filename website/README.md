# Prodara Website

The [prodara.dev](https://prodara.dev) marketing and documentation site. Built with Angular 21 + Tailwind CSS + SSR.

## Development

```bash
ng serve
```

Open `http://localhost:4200/`. The app reloads on file changes.

## Building

```bash
ng build
```

Production artifacts are written to `dist/`.

## Testing

```bash
ng test
```
